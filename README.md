# AlgorithmPython

Python으로 알고리즘 문제를 풀이한 코드를 모아둔 저장소입니다.

## 구성

| 폴더 | 플랫폼 | 설명 |
|------|--------|------|
| `Baekjoon/` | [백준 온라인 저지](https://www.acmicpc.net/) | 일반 문제 및 삼성SW 대비 |
| `Codetree/` | [코드트리](https://www.codetree.ai/) | 삼성SW 기출 |

## Baekjoon 폴더 구조

```
Baekjoon/
├── src/              # 일반 문제

└── 한번에못푼문제/    # 복습 필요 문제
```
